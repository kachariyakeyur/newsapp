# 📰 NewsApp

A React-based News Application that fetches the latest headlines using a custom backend API built with Node.js and Express.

## 🚀 Live Demo

- **Frontend (Vercel):** https://newsapp-tau-tan.vercel.app/
- **Backend (Render):** https://newsapp-backend-cstw.onrender.com/

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Bootstrap / CSS

### Backend
- Node.js
- Express.js
- CORS

### Deployment
- Vercel (Frontend)
- Render (Backend)
